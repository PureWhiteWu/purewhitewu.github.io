这两天又发现一个问题，就是hexo的html、css、js都是没有压缩过的，这样是很不好的，因为会浪费服务器的资源 & 让用户的下载速度变慢。

于是继续使用Google大法搜索有没有解决方案，然后看到了前辈已经遇到过这种问题了，原文链接放在了最后，这里整理一下过程。

为什么我们需要压缩

通过压缩可以明显的减少静态资源的体积，加快响应速度。

使用gulp

我们使用了 gulp 来定义要执行的压缩任务，而执行真正压缩的是如下几个模块，见名知意：

  • clean-css
  • htmlclean
  • htmlmin
  • imagemin
  • uglify-js
